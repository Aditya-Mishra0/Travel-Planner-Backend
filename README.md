## Backend (Spring Boot)

The backend is a robust RESTful API that handles business logic, security, and external service orchestration.

**Key Features**
- Data Persistence: Uses Spring Data JPA with a MySQL database.

- Security: Spring Security implementation for user authentication.

- Media Handling: Integration with Cloudinary API for cloud-based image storage.

- External API Bridge: Fetches real-time city data from external location APIs.

**Tech Stack**
- Language: Java 17 (OpenJDK)

- Framework: Spring Boot 4.0.1

- ORM: Hibernate / JPA

- Database: MySQL 8.0
